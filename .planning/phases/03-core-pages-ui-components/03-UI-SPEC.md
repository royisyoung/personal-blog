## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage                     |
| ----- | ----- | ------------------------- |
| xs    | 4px   | Icon gaps, inline padding |
| sm    | 8px   | Compact element spacing   |
| md    | 16px  | Default element spacing   |
| lg    | 24px  | Section padding           |
| xl    | 32px  | Layout gaps               |
| 2xl   | 48px  | Major section breaks      |
| 3xl   | 64px  | Page-level spacing        |

Exceptions: Theme toggle icon button requires 44px touch target for accessibility.

## Color

Following shadcn neutral/zinc preset with 60/30/10 principle:

| Role            | Value               | Usage                                                       |
| --------------- | ------------------- | ----------------------------------------------------------- |
| Dominant (60%)  | zinc-50 / zinc-950  | Background, surfaces                                        |
| Secondary (30%) | zinc-100 / zinc-900 | Cards, sidebar, nav                                         |
| Accent (10%)    | zinc-900 / zinc-100 | Links, active nav, pagination current page, category badges |
| Destructive     | red-500 / red-400   | Not used in this phase                                      |

Accent reserved for:

- Interactive links (post titles, navigation links)
- Current page indicator in pagination
- Category/tag badges on post list
- Active navigation link styling
