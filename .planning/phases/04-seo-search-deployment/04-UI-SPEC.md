## Spacing Scale

Declared values (multiples of 4, 8-point grid):

| Token | Value | Usage                     |
| ----- | ----- | ------------------------- |
| xs    | 4px   | Icon gaps, inline padding |
| sm    | 8px   | Compact element spacing   |
| md    | 16px  | Default element spacing   |
| lg    | 24px  | Section padding           |
| xl    | 32px  | Layout gaps               |
| 2xl   | 48px  | Major section breaks      |
| 3xl   | 64px  | Page-level spacing        |

Exceptions: Search modal interactive elements require 44px minimum click area for accessibility (confirmed by user).

## Color

Uses shadcn default neutral color palette with CSS variables (configured by shadcn):

| Role            | Value                | Usage                                                                   |
| --------------- | -------------------- | ----------------------------------------------------------------------- |
| Dominant (60%)  | `background`         | Page backgrounds, main surfaces                                         |
| Secondary (30%) | `card` / `secondary` | Cards, navigation, modal content background                             |
| Accent (10%)    | `primary`            | Search button, search input focus border, search result highlight links |

Accent reserved for:

- Search submit button

## Interaction Contracts

> From upstream CONTEXT.md decisions:

1. **Search Trigger**: Search icon button in site header (accessible from any page)
2. **Search Modal**: Clicking search icon opens modal dialog with search input focused
3. **Search Behavior**: User types query, results update asynchronously without page navigation
4. **Close**: Click outside modal or press Escape to close
5. **Result Interaction**: Clicking a search result closes modal and navigates to the post
