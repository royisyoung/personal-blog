## Spacing Scale

Declared values (8-point scale, multiples of 4):

| Token | Value | Usage                     |
| ----- | ----- | ------------------------- |
| xs    | 4px   | Icon gaps, inline padding |
| sm    | 8px   | Compact element spacing   |
| md    | 16px  | Default element spacing   |
| lg    | 24px  | Section padding           |
| xl    | 32px  | Layout gaps               |
| 2xl   | 48px  | Major section breaks      |
| 3xl   | 64px  | Page-level spacing        |

Exceptions: Back to Top button touch target at 48px (48x48), which conforms to 8-point scale.

## Color

| Role | Value (light) | Value (dark) | Usage |
| --- | --- | --- | --- |
| Dominant (60%) | `--background oklch(1 0 0)` | `--background oklch(0.145 0 0)` | Page background, main surfaces |
| Secondary (30%) | `--card oklch(1 0 0)` / `--secondary oklch(0.97 0 0)` | `--card oklch(0.205 0 0)` / `--secondary oklch(0.269 0 0)` | Cards, sidebar, navigation, TOC background |
| Accent (10%) | `--primary oklch(0.205 0 0)` | `--primary oklch(0.922 0 0)` | Reading progress bar, interactive highlights, active TOC heading, copy button checkmark |
| Destructive | `--destructive oklch(0.577 0.245 27.325)` | `--destructive oklch(0.704 0.191 22.216)` | Errors (no destructive actions in this phase) |

Accent reserved for:

- Reading progress bar fill
- Active/Current table of contents heading highlighting
- Copy button success checkmark icon
- Back to Top button icon/background

_Uses existing shadcn radix-nova color palette from globals.css_
