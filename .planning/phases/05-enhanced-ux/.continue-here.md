<completed_work>

**Features discussed and decisions made:**

1. **Reading progress bar** (UX-01):
   - Position: Thin bar at viewport top
   - Height: Thin (2-4px)
   - Visibility: Always visible while scrolling
   - Behavior: Updates width dynamically based on scroll position

2. **Code block copy button** (UX-03):
   - Position: Fixed at top-right corner of each code block
   - Interaction: Click to copy → icon changes to checkmark for 2 seconds then reverts
   - Styling: Follows site theme (dark/light)

3. **Auto-generated Table of Contents** (UX-04):
   - Position: Sidebar at the beginning of article (left side)
   - Scope: Only includes H2 and H3 headings
   - Behavior: Highlights current heading while scrolling
   - Positioning: Sticky/fixed to top when scrolling

4. **Related posts** (UX-05):
   - Quantity: Show 2-3 related posts
   - Sorting: By number of shared tags (more shared = more relevant)

5. **Back to top button** (UX-02):
   - Position: Fixed at bottom-right corner of screen
   - Visual: Circular button with upward arrow icon
   - Visibility: Always visible
   - Animation: Smooth scroll animation when clicked

</completed_work>
