  <action>
1. Create `lib/toc.ts` with the following exports:

- Type `Heading`: `{ depth: 2 | 3; value: string; id: string }`
- Function `slugifyHeading(text: string): string` - converts heading text to URL-safe anchor id:
  - Lowercase, replace spaces with hyphens, remove non-alphanumeric except hyphens
  - Handle collisions by appending counter if same id already exists
- Function `extractHeadings(node: any, headingIds: Set<string>): Heading[]` - recursively traverses Hast AST, collects only h2 and h3
- Rehype plugin function `rehypeExtractHeadings` - attaches extracted headings to `doc.data` so Contentlayer can read it

Notes for collision handling: when same slug encountered multiple times, add -1, -2 suffix to ensure uniqueness.

2. Add `headings` computed field to `contentlayer.config.ts`:

```typescript
headings: {
  type: 'list',
  of: {
    type: 'object',
    fields: {
      depth: { type: 'number', required: true },
      value: { type: 'string', required: true },
      id: { type: 'string', required: true },
    },
  },
  resolve: (doc) => doc._raw.data?.headings || [],
}
```
